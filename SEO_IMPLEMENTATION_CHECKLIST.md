## Troubleshooting Guide

### If pages not indexing:
1. Check Google Search Console
2. Submit sitemap.xml
3. Manually request indexing for new URLs
4. Check robots.txt for allow rules

### If rankings not improving:
1. Check for indexing errors
2. Verify keyword difficulty isn't too high
3. Build 2-3 quality backlinks
4. Create more supporting content
5. Improve page load speed

### If low traffic from blog:
1. Check social media sharing
2. Add more internal links
3. Create related blog posts
4. Optimize meta descriptions
5. Add schema markup
